# Chip8 interpreter
## Usage
`cargo run <path to chip8 file>`
## Keypad mapping
```
1 2 3 4 -> 1 2 3 C

Q W E R -> 4 5 6 D

A S D F -> 7 8 9 E

Z X C V -> A 0 B F
```

## Chip8 architecture reference
http://devernay.free.fr/hacks/chip8/C8TECH10.HTM

## Where to find roms
Roms from [this](https://github.com/kripod/chip8-roms) repo should work
